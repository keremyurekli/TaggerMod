# TaggerMod
Simple way to tag/mark blocks

## **Keybindings by default:**                
  >Tag/Mark the blocks: **Z**                          
  >Enable/Disable tracer: **X**                   
  >Clear tagged blocks: **G**              
 
You can change them in settings.
